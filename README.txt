Prerequisite: Python 3

Step 1: Open CLI (with python) and run 
      "pip3 install mysql-connector-python" or "pip install mysql-connector-python"
      "python3 db_mysql.py" or "python db_mysql.py"
Step 2: There will be a folder named "dataset" created in the working directory.
Step 3: Download this app https://tzutalin.github.io/labelImg/
Step 4: Run app (labelImg.exe in windows).
Step 5: In the GUI, click "Open Dir" and select the "dataset" directory.
Step 6: You can see all of the images are labelled.

Thank you!
